输入组件 widgets

widgets 是用户与shiny交互的web元素，为用户提供一种将消息发送到shiny应用的方法。下面介绍如何向shiny应用添加这些小组件。

用户通过widgets输入数据，参数等，当用户通过小部件更改时，相应的参数数据也会发生改变，来进行接下来的计算。

要将widgets添加到应用中，需要将widgets放在ui程序面板函数，例如sidebarPanel,mainPanel中。

对应的代码如下

library(shiny)

# Define UI ----
ui <- fluidPage(
  titlePanel("Basic widgets"),
  
  fluidRow(
    
    column(3,
           h3("Buttons"),
           actionButton("action", "Action"),
           br(),
           br(), 
           submitButton("Submit")),
    
    column(3,
           h3("Single checkbox"),
           checkboxInput("checkbox", "Choice A", value = TRUE)),
    
    column(3, 
           checkboxGroupInput("checkGroup", 
                              h3("Checkbox group"), 
                              choices = list("Choice 1" = 1, 
                                             "Choice 2" = 2, 
                                             "Choice 3" = 3),
                              selected = 1)),
    
    column(3, 
           dateInput("date", 
                     h3("Date input"), 
                     value = "2014-01-01"))   
  ),
  
  fluidRow(
    
    column(3,
           dateRangeInput("dates", h3("Date range"))),
    
    column(3,
           fileInput("file", h3("File input"))),
    
    column(3, 
           h3("Help text"),
           helpText("Note: help text isn't a true widget,", 
                    "but it provides an easy way to add text to",
                    "accompany other widgets.")),
    
    column(3, 
           numericInput("num", 
                        h3("Numeric input"), 
                        value = 1))   
  ),
  
  fluidRow(
    
    column(3,
           radioButtons("radio", h3("Radio buttons"),
                        choices = list("Choice 1" = 1, "Choice 2" = 2,
                                       "Choice 3" = 3),selected = 1)),
    
    column(3,
           selectInput("select", h3("Select box"), 
                       choices = list("Choice 1" = 1, "Choice 2" = 2,
                                      "Choice 3" = 3), selected = 1)),
    
    column(3, 
           sliderInput("slider1", h3("Sliders"),
                       min = 0, max = 100, value = 50),
           sliderInput("slider2", "",
                       min = 0, max = 100, value = c(25, 75))
    ),
    
    column(3, 
           textInput("text", h3("Text input"), 
                     value = "Enter text..."))   
  )
  
)

# Define server logic ----
server <- function(input, output) {
  
}

# Run the app ----
shinyApp(ui = ui, server = server)

函数

通常每一个组件有多个参数，前两个参数为 - 部件的名字，用户不会看到，server函数调用 - label, 用户看到的名字，用于提示用户识别这个部件的意义

常用函数

actionButton 点击次数
checkboxGroupInput 一组复选框，可以多选
checkboxInput 单个复选框，选为True,不选为False
dateInput 选择日期
dateRangeInput 日期范围
fileInput 输入文件
helpText 帮助文本
numericInput 数字输入
radioButtons 单选按钮
selectInput 选择
sliderInput 滑杆输入，通过value可以控制单值或两个值
submitButton 提交按钮
textInput 输入文本
