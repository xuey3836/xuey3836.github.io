交互式输出

接下来讲述如何在shiny应用程序中生产反应式输出，当用户更改widget值时，在用户界面就会有对应的输出。

新的shiny应用需要一个新的目录，通常是创建一个新的文件夹，然后新建一个app.R 或者 一个ui.R和server.R。reactive 输出通过下面两步完成 - 在ui里加入R对象，函数名为 * Output, 这里的添加方式如添加小部件widgets(*Input)一样, 需要将其放在 *Panel函数中 - 告诉shiny如何在server功能中构建对象, 如果构建对象的代码调用小部件的值，则该对象可以根据小部件的值相应发生变化

render* 函数和 *Output函数一起使用可以将R对象输出到用户界面。

在用户界面中添加R对象

Shiny提供了一系列功能，这些功能可以将R对象转换为用户界面的输出。 每个函数都会创建特定类型的输出，而且在server函数都有相应对应函数，如下表所示。你可以在将这些output像添加小部件widgets一样添加到用户界面中，将这些output放在sidebarPanel或mainPanel中。

输出函数(ui) 输出对象 render 函数(server) packages
dataTableOutput 数据表 renderDataTable shiny
htmlOutput 网页 renderUI shiny
imgOutput 图片 renderImage shiny
plotOutput 作图 renderPlot shiny
tableOutput 表格 renderTable shiny
textOutput 文本 renderText shiny
uiOutput 网页 renderUI shiny
verbatimTextOutput 文本 renderPrint shiny
plotlyOutput 交互图 renderPlotly plotly

renderPrint 与 renderText 的区别

renderPrint(
  expr,
  env = parent.frame(),
  quoted = FALSE,
  width = getOption("width"),
  outputArgs = list()
)

renderText(
  expr,
  env = parent.frame(),
  quoted = FALSE,
  outputArgs = list(),
  sep = " "
)

renderPrint打印expr的结果，与R中的print相当，而renderText是把他们整合一个string输出，与R中的cat相当。

renderCachePlot

renderCachePlot 与renderPlot 功能一样，只是能够将已经出现的plot存储下来，再次调用时不用重新计算，直接调用cache下来的图。例如给定直方图的bins作为小组件的输入，可以画出不同的图，使用renderCachePlot时，比如第一次选用bins = 2, 图像显示出来，第二次选用bins = 4, 又再server通过renderCachePlot计算在用户界面显示一张图，但当再次选用bins =2时，此时可以不用计算，直接调用缓存中的图在用户界面显示。

在shiny 1.6.0之后，官方建议使用renderPlot 结合 bindCache 来代替renderCachePlot。参考链接

执行路径

在UI中创建了一个output，你需要告诉shiny怎样去创建它，这时候你需要在server函数里创建对应的对象，server所有需要输出在output列表里，以 “output中的elem” 作为这个对象的名字，元素名称elem应与UI中创建的输出函数的名称相匹配, 如下面例子中，执行路径如下所述。

  • 改变siderInput，即改变input$range
  • renderText 读取input$range, 更新output列表中的elem
  • textOutput读取elem,最终输出到用户界面上
ui <- fluidPage(
  titlePanel("reactive example"),
  sidebarLayout(
    sidebarPanel(

      sliderInput(inputId = "bins",
                  label = "Number of bins:",
                  min = 1,
                  max = 50,
                  value = 30)
    ), 
    mainPanel(
      textOutput("elem")
    )
  ))
  server <- function(input, output) {
    output$elem <- renderText({ 
      paste("The input is", input$bins)
    })
  }
  shinyApp(ui, server)
