Shiny 简介

Shiny是一个R软件包，可轻松从R直接构建交互式Web应用程序。您可以在网页上托管独立应用程序，也可以将其嵌入R Markdown文档中或构建仪表板。 还可以使用CSS主题，htmlwidgets和JavaScript操作扩展Shiny应用程序。

它将R的计算能力和Web的交互性结合在一起，而且不需要太多的网页开发知识就可轻松完成一个Web交互应用。

## 安装shiny包
install.packages('shiny')

例子

shiny包里有一些例子，每一个例子都是独立的Shiny应用。如下面这个 Hello Shiny 的例子是faithful数据下的直方图，用户可以通过滑杆改变直方图的箱数，应用程序会相应的画出不同的直方图。

library(shiny)
runExample("01_hello")

可以通过shiny包的安装目录找到相应的程序所在目录，一般是”C:/Program Files/R/R-4.0.2/library/shiny/examples/01_hello”，让我们来看一下一个应用包含哪些文件，

  • app.R：应用程序
  • Readme.md：程序注释，如果display.mode = ‘showcase’ 时展示
  • DESCRIPTION: 应该介绍

应用程序结构 app.R

每一个Shiny应用程序都有一样的结构，app.R包括ui(模块布局)和server(R语言计算). 如下所示，

library(shiny)

# See above for the definitions of ui and server
ui <- ... 
server <- ...
shinyApp(ui = ui, server = server)

创建一个应用的步骤， - 创建一个新的目录，例如命名为my_app - 写app.R程序,并保存到my_app目录下

此时可以运行这个应用, 在Rstudio中点击Run APP图标就可以，等同于运行下面这个命令

# display.mode = 'showcase', 展示app.R代码和Readme内容
# display.mode = 'normal', 只展示ui内容
runApp("my_app", display.mode = 'showcase')

例子程序详解

  • fluidPage, 函数，包裹排版函数，自动调整页面
  • titlePanel(标题面板),sidebarLayout(侧边栏排版)
  • sidebarPanel(侧边栏面板)，mainPanel(主面板：直方图)
  • sliderInput(滑动杆，输入)，server 用 input$bins调用
  • plotOutput，调用server里的output$distplot,
  • 运行流程，server函数读取ui函数的input$bins，然后画图，ui的plotOutput调用server函数的output$plot进行展示。

ui function

library(shiny)

# 定义直方图应用的UI程序 ----
ui <- fluidPage(

  # App 标题 ----
  titlePanel("Hello Shiny!"),

  # Sidebar layout with input and output definitions ----
  sidebarLayout(

    # Sidebar panel for inputs ----
    sidebarPanel(

      # Input: Slider 直方图的bins ----
      sliderInput(inputId = "bins",
                  label = "Number of bins:",
                  min = 1,
                  max = 50,
                  value = 30)

    ),

    # Main panel for displaying outputs ----
    mainPanel(

      # Output: Histogram ----
      plotOutput(outputId = "distPlot")
      ## 对应server 里的输出

    )
  )
)

server function

# Define server logic required to draw a histogram ----
server <- function(input, output) {

  # Histogram of the Old Faithful Geyser Data ----
  # 输入直方图的bins
  #
  # 1. 是连锁反应的，根据输入的不同自动反应，有交互的效果
  #    当输入不同的bins,会画出不同的直方图
  # 2. 输出类型是直方图
  output$distPlot <- renderPlot({

    x    <- faithful$waiting
    bins <- seq(min(x), max(x), length.out = input$bins + 1)

    hist(x, breaks = bins, col = "#75AADB", border = "white",
         xlab = "Waiting time to next eruption (in mins)",
         main = "Histogram of waiting times")

    })

}
